V# Documentation

V# is a JavaScript utility project that provides enhanced functionality for console logging, HTTP requests, and JSON handling. It's designed to make common web development tasks more efficient and less error-prone.

V# offers a clean API with consistent error handling, informative logging, and simplified HTTP operations.

Project Structure

V# is organized into modular JavaScript files:

assets/
  └── functions/
      ├── console.js    # Enhanced console logging
      ├── fetch.js      # HTTP request utilities
      └── json.js       # JSON handling utilities

Quick Start

To use V# in your project, simply import the modules you need:

// ES Modules
import { print } from './assets/functions/console.js';
import { fetch } from './assets/functions/fetch.js';
import { json } from './assets/functions/json.js';

// CommonJS
const { print } = require('./assets/functions/console.js');
const { fetch } = require('./assets/functions/fetch.js');
const { json } = require('./assets/functions/json.js');

Console Module

The Console module (exported as print) enhances JavaScript's native console functionality by adding prefixes to different log types and providing access to all console methods through a unified interface.

print.log(message)

Logs a message to the console.

message: Any - The message to log
Returns: void 
print.log("I love V#");
// Prints to console: "I love V#"

print.debug(message)

Logs a debug message with a "[DEBUG]" prefix.

message: Any - The debug message to log
Returns: void 
print.debug("V# is developed by lunary-net in JavaScript");
// Prints to console: "[DEBUG] V# is developed by lunary-net in JavaScript"

print.error(message)

Logs an error message with an "[ERROR]" prefix.

message: Any - The error message to log
Returns: void 
print.error("Error 404. Not Found");
// Prints to console: "[ERROR] Error 404. Not Found"

print.info(message)

Logs an informational message with an "[INFO]" prefix.

message: Any - The info message to log
Returns: void 
print.info("Logged in as lunary-net");
// Prints to console: "[INFO] Logged in as lunary-net"

print.warn(message)

Logs a warning message with a "[WARNING]" prefix.

message: Any - The warning message to log
Returns: void 
print.warn("This feature will be deprecated soon");
// Prints to console: "[WARNING] This feature will be deprecated soon"

The console module also includes all other native console methods:

  • print.trace() - Outputs a stack trace
  • print.dir(obj) - Displays an interactive list of object properties
  • print.table(data) - Displays tabular data as a table
  • print.count(label) - Counts number of times a label is logged
  • print.time(label), print.timeEnd(label) - Timing operations
  • print.group(), print.groupEnd() - Creates grouped log messages
  • And more...

Fetch Module

The Fetch module provides enhanced HTTP request functionality with built-in error handling, automatic JSON parsing, and specialized methods for different HTTP verbs.

fetch.get(url, options)

Performs a GET request and automatically parses the JSON response.

url: String - The URL to request
options: Object (optional) - Fetch options
Returns: Promise<any> - Parsed JSON response 
// Basic GET request
const data = await fetch.get('https://api.example.com/users');

// With custom headers
const userData = await fetch.get('https://api.example.com/user/1', {
  headers: { 'Authorization': 'Bearer token123' }
});

fetch.post(url, data, options)

Performs a POST request with JSON body and automatically parses the JSON response.

url: String - The URL to request
data: Object - The data to send in request body
options: Object (optional) - Fetch options
Returns: Promise<any> - Parsed JSON response 
// Create a new user
const newUser = await fetch.post('https://api.example.com/users', {
  name: 'John Doe',
  email: 'john@example.com'
});

The fetch module also includes similar methods for other HTTP methods:

  • fetch.put(url, data, options) - For PUT requests
  • fetch.delete(url, options) - For DELETE requests
  • fetch.patch(url, data, options) - For PATCH requests

fetch.getText(url, options)

Performs a GET request and returns the text response instead of parsing as JSON.

url: String - The URL to request
options: Object (optional) - Fetch options
Returns: Promise<string> - Text response 
// Get HTML content
const htmlContent = await fetch.getText('https://example.com');

fetch.getBlob(url, options)

Performs a GET request and returns the response as a blob (for files, images, etc.).

url: String - The URL to request
options: Object (optional) - Fetch options
Returns: Promise<Blob> - Blob response 
// Download an image
const imageBlob = await fetch.getBlob('https://example.com/logo.png');
const imageUrl = URL.createObjectURL(imageBlob);
document.getElementById('myImage').src = imageUrl;

fetch.uploadFile(url, fileData, fileName, options)

Uploads a file using FormData.

url: String - The URL to upload to
fileData: File|Blob - The file data to upload
fileName: String (optional) - The name to use for the file field (default: 'file')
options: Object (optional) - Fetch options
Returns: Promise<any> - Parsed JSON response 
// Upload a file from an input element
const fileInput = document.getElementById('fileInput');
const file = fileInput.files[0];
const result = await fetch.uploadFile('https://api.example.com/upload', file);

JSON Module

The JSON module provides a convenient wrapper around JavaScript's native JSON functionality.

json.stringify(value)

Converts a JavaScript value to a JSON string.

value: Any - The value to convert to JSON
Returns: String - JSON string representation 
const user = { name: 'John', age: 30 };
const jsonString = json.stringify(user);
// '{"name":"John","age":30}'

json.parse(text)

Parses a JSON string and returns the resulting JavaScript value.

text: String - The JSON string to parse
Returns: Any - The parsed JavaScript value 
const jsonString = '{"name":"John","age":30}';
const user = json.parse(jsonString);
// { name: 'John', age: 30 }

Console Module Examples

Here's a more comprehensive example of using the Console module:

const { print } = require('./assets/functions/console');

// Basic logging with prefixes
print.debug("V# is developed by lunary-net in JavaScript");
print.error("Error 404. Not Found");
print.info("Logged in as lunary-net");
print.log("I love V#");
print.warn("This feature will be deprecated soon");

// Advanced logging
const user = { name: "lunary-net", role: "admin", id: 12345 };
print.dir(user);

// Tabular data
const users = [
  { name: "lunary-net", role: "admin", id: 12345 },
  { name: "user123", role: "member", id: 67890 }
];
print.table(users);

// Grouping related logs
print.group("User Authentication");
print.log("Starting authentication process");
print.log("Checking credentials...");
print.log("Authentication successful");
print.groupEnd();

// Timing operations
print.time("dataFetch");
// ... some operation ...
print.timeLog("dataFetch", "Data partially loaded");
// ... more operations ...
print.timeEnd("dataFetch");

Fetch Module Examples

Here's a comprehensive example of using the Fetch module:

const { fetch } = require('./assets/functions/fetch');
const { print } = require('./assets/functions/console');

async function fetchExamples() {
  try {
    // GET request with JSON response
    const users = await fetch.get('https://api.example.com/users');
    print.info(`Retrieved ${users.length} users`);
    
    // POST request with JSON body
    const newUser = await fetch.post('https://api.example.com/users', {
      name: 'Jane Doe',
      email: 'jane@example.com'
    });
    print.info(`Created new user with ID: ${newUser.id}`);
    
    // PUT request to update a resource
    const updatedUser = await fetch.put(`https://api.example.com/users/${newUser.id}`, {
      name: 'Jane Smith',
      email: 'jane.smith@example.com'
    });
    print.info(`Updated user: ${updatedUser.name}`);
    
    // GET text content
    const htmlContent = await fetch.getText('https://example.com');
    print.info(`Retrieved ${htmlContent.length} bytes of HTML`);
    
    // GET a file as blob
    const imageBlob = await fetch.getBlob('https://example.com/logo.png');
    print.info(`Retrieved image of size: ${imageBlob.size} bytes`);
    
    // Upload a file (example with file from input)
    // const fileInput = document.getElementById('fileInput');
    // const file = fileInput.files[0];
    // const uploadResult = await fetch.uploadFile('https://api.example.com/upload', file);
    // print.info(`Uploaded file: ${uploadResult.filename}`);
    
    // DELETE a resource
    await fetch.delete(`https://api.example.com/users/${newUser.id}`);
    print.info(`Deleted user with ID: ${newUser.id}`);
    
  } catch (error) {
    print.error(`API Error: ${error.message}`);
  }
}

fetchExamples();

JSON Module Examples

Here's an example of using the JSON module:

const { json } = require('./assets/functions/json');
const { print } = require('./assets/functions/console');

// Convert JavaScript object to JSON string
const user = {
  name: 'John Doe',
  age: 30,
  email: 'john@example.com',
  roles: ['user', 'admin'],
  metadata: {
    lastLogin: new Date(),
    preferences: {
      theme: 'dark',
      notifications: true
    }
  }
};

const jsonString = json.stringify(user);
print.info('Converted object to JSON string:');
print.log(jsonString);

// Parse JSON string back to JavaScript object
const parsedUser = json.parse(jsonString);
print.info('Parsed back to JavaScript object:');
print.dir(parsedUser);

// Note: The Date object becomes a string when converted to JSON
print.info(`Original date type: ${typeof user.metadata.lastLogin}`); // object
print.info(`Parsed date type: ${typeof parsedUser.metadata.lastLogin}`); // string
